Chat Desktop Application in Java

In my OOP class this semester I had the opportunity to work on a project in Java in pairs. The objective was to create a desktop chat app with a distributed architecture.


The source code is available here (this is my academic git account). This is a Gradle project using sockets, a servlet (in another repo) and JavaFx for the UI. The features include a nickname change option, a message history or the possiblity to send files and images.
