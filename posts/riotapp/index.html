Riot Summonner Info app in Python

It not very useful, very ugly and has probably already been coded 100 times but here it is : my "Riot Summonner Info App"


I coded it in Python using TKinter (I know, it's old and no one uses it but I wanted to learn). I also used the Riot API and CommunityDragon CDN for the assets. The source code is available here. To sum it up, this app allows you to search for a summoner with their name and it will display infos such as :

  • Their level and their rank (if unranked you get a cute Jigglypuff)
  • Their summoner icon
  • Their favorite champions with their masteries
  • Whether the player is in game or not and if yes, you get the champion they are using

The result is really (really) bad looking (what was I expecting using Tkinter) but it was really fun to create. Maybe I will try to create an in-game app later.
